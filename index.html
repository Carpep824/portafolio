Mis Conocimientos

Java

Ideal para desarrollar aplicaciones móviles y de escritorio, especialmente cuando se necesita que funcionen en diferentes plataformas.

Python

Perfecto para la creación de aplicaciones web debido a su simplicidad y versatilidad.

HTML, CSS & JavaScript

Son las tecnologías básicas para crear páginas web: HTML organiza el contenido, CSS se encarga del diseño y JavaScript hace las páginas interactivas.

MySQL

Usado para administrar bases de datos en proyectos web, ayudando a almacenar y recuperar datos rápidamente.

PostgreSQL

Diseñado para gestionar grandes bases de datos con requisitos complejos, ideal para aplicaciones que necesitan manejar mucha información.

Arduino

Una plataforma que permite crear dispositivos electrónicos y proyectos interactivos, ideal para prototipos y proyectos de IoT.
